# 21+ 打卡APP

一个轻量级的习惯打卡应用，帮助你坚持21天养成好习惯。

---

## ✨ 功能特性

- **习惯管理**：添加、编辑、删除习惯
- **一键打卡**：快速完成每日打卡
- **进度追踪**：21天进度环可视化显示
- **连续打卡**：自动计算连续坚持天数
- **打卡日历**：月度日历查看历史记录
- **数据备份**：支持JSON导出/导入
- **成就激励**：完成21天挑战显示庆祝动画

## 🎨 设计特点

- 深色主题，护眼舒适
- 简洁直观的用户界面
- 流畅的动画效果
- 专为手机端优化

## 🔧 使用说明
### 访问地址
https://ZiChen-hub.github.io/21plus-habit-/

### 添加习惯
1. 点击底部的 **添加习惯** 按钮
2. 输入习惯名称（如：每日健身、戒烟、阅读等）
3. 选择一个图标
4. 点击 **创建习惯**

### 打卡
1. 在首页点击习惯旁边的 **打卡** 按钮
2. 打卡成功后按钮会变成绿色的 ✓

### 查看详情
1. 点击习惯卡片可以查看详情
2. 查看连续天数、进度和打卡日历

### 数据备份
1. 点击底部导航栏的 **设置**
2. 点击 **导出数据** 保存备份文件
3. 点击 **导入数据** 恢复备份
